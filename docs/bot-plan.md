# План створення Telegram-бота — Telegram Saver

## Що будуємо

Telegram-бот `@AI_hol228MBot` — це окремий Node.js сервер,
який працює 24/7 і приймає повідомлення від користувачів.

---

## Крок 1 — Підготовка папки бота

```
telegram-saver-bot/
├── src/
│   ├── index.ts        ← точка входу, запуск бота
│   ├── commands.ts     ← команди /start, /help, /list
│   └── handlers.ts     ← обробка повідомлень
├── .env                ← токен бота і ключі Supabase
├── package.json
└── tsconfig.json
```

**Технології:**
- `grammY` — бібліотека для Telegram Bot API
- TypeScript — мова програмування
- Supabase — та сама база даних що і в сайту

---

## Крок 2 — Команди бота

| Команда | Що робить |
|---------|-----------|
| `/start` | Вітання + інструкція. Зберігає telegram_user_id в базу |
| `/help` | Список можливостей бота |
| `/list` | Останні 5 збережень користувача |
| `/login` | Надсилає одноразовий код для входу на сайт |

---

## Крок 3 — Авторизація через бота (фікс входу на сайт)

**Проблема:** Telegram Login Widget не надсилає підтвердження.

**Рішення — вхід через код від бота:**

```
1. Користувач відкриває telegram-saver.vercel.app/login
2. Вводить свій @username в поле
3. Натискає "Отримати код"
4. НАШ БОТ надсилає 6-значний код в Telegram
5. Користувач вводить код на сайті
6. Сайт перевіряє код і дає доступ
```

Для цього потрібно:
- Бот має бути запущений (цей план)
- Користувач має написати `/start` боту хоч раз
- На сайті — нова форма логіну

---

## Крок 4 — Зберігання повідомлень (Етап 4)

Бот приймає:
- **Текст** → зберігає в `saved_items` з `type="text"`
- **Посилання** → парсить сторінку → зберігає з `type="link"`
- **Фото** → завантажує в Supabase Storage → `type="photo"`
- **Forward** → зберігає з `type="forward"` + дані каналу

---

## Крок 5 — Деплой на Railway

Railway — хостинг для Node.js процесів (не сайтів).
Бот має працювати постійно — Railway забезпечує це за ~$5/міс.

**Кроки деплою:**
1. Реєстрація на railway.app (через GitHub)
2. New Project → Deploy from GitHub Repo
3. Додати змінні середовища (.env)
4. Railway сам будує і запускає бота
